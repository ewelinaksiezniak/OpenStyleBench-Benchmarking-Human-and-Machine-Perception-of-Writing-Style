# OpenStyleBench — Benchmarking Human and Machine Perception of Writing Style

**OpenStyleBench** is a research framework for studying how writing style is perceived by humans and machine learning models under controlled conditions.

The project investigates the gap between human and computational representations of style by generating semantically equivalent texts with systematically varied stylistic features. It enables direct comparison of human judgments and transformer-based model performance under targeted stylistic perturbations.

The primary goal is to identify which stylistic cues can be modified to disrupt automated authorship recognition while preserving human-perceived style, coherence, and meaning.

## Key Components
- Controlled dataset of semantically equivalent stylistic variants  
- Evaluation pipeline for transformer-based models  
- Human evaluation protocols (behavioral experiments)  
- Tools for analyzing stylistic robustness and model generalization  

## Research Focus
OpenStyleBench is designed as a **scientific and open research resource**, supporting work on:
- Stylometry and authorship analysis  
- Human–model perception differences  
- Privacy-preserving text transformation  
- Human-aligned and trustworthy language technologies  

All datasets, code, and evaluation protocols are released as open resources to support reproducible research.
